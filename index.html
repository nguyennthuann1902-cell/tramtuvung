⚙️ CẤU HÌNH FIREBASE

Dán đoạn mã cấu hình Firebase Web SDK của bạn vào đây. Hệ thống sẽ tự động đồng bộ hóa tất cả tiến trình của bạn lên Firebase Realtime Database.

💡 Thiết lập trong 1 phút:
1. Tạo project trên Firebase Console.
2. Bật dịch vụ Realtime Database (quy tắc đọc/ghi: true hoặc có phân quyền).
3. Vào Project settings -> Tạo Web App -> copy code config và dán vào đây.
4. Bật Authentication -> Thêm Google Sign-in provider.
